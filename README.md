# 🚀 OpportunityAI

## Problem Statement

Students often miss hackathons, internships, scholarships, competitions, and career opportunities because they are scattered across multiple platforms. Finding opportunities that match their skills, interests, academic background, and location is difficult.

## Solution

OpportunityAI is an AI-powered web platform that helps students discover personalized opportunities using AI profile analysis and smart matching.<img width="1887" height="966" alt="Screenshot 2026-06-20 102214" src="https://github.com/user-attachments/assets/469df7cc-af32-4a9c-9a1d-9aab8682af31" />
