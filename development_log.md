# Project Objective

To build an AI-powered platform that helps students discover hackathons, internships, scholarships, competitions, and career opportunities based on their skills, interests, academic profile, and preferred location.

---

# Important AI Prompts Used

### Prompt 1
Generate a modern and responsive React application with Dark and Light theme for an AI-powered student opportunity platform.

### Prompt 2
Build an attractive dashboard with profile management, opportunity cards, search, filters, and bookmarking.

### Prompt 3
Generate personalized AI career guidance including:
- Skill Gap Analysis
- Career Summary
- Learning Roadmap
- Resume Suggestions
- Interview Preparation Tips

### Prompt 4
Improve the user interface using Tailwind CSS and modern UI design principles.
